# QueueSmart Front-End Prototype

QueueSmart is a smart queue management UI/UX prototype for Assignment 2. This version is front-end only and uses mock data/placeholders. It supports both regular users and administrators.

## What is included

- Login screen with email/password validation
- Registration screen with required fields, email validation, password length check, and role selection
- User dashboard with current queue status, active services, and notifications
- Join Queue screen with service selection, estimated wait time, join queue, and leave queue UI
- Queue Status screen with position, estimated wait time, and status updates
- History screen with past queue records
- Admin dashboard with services, queue lengths, and quick actions
- Service Management screen with required fields, max length, number input, and priority level
- Queue Management screen with mock queue list, reorder/remove buttons, and serve-next simulation
- In-app notifications
- Custom SVG logo stored in the `assets` folder

## Technologies used

- HTML5
- CSS3
- JavaScript

These were chosen because the assignment focuses on UI/UX and front-end implementation. Plain HTML, CSS, and JavaScript keep the project simple, easy to run, and easy for every group member to understand.

## How to run

Open `index.html` in a web browser.

No backend, server, database, or package installation is required.
